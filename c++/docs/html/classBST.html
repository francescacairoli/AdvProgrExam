Insert a new node in the binary search tree.

Parameters
dThe key,value pair to be inserted.
Returns
True if the insert operation is successful, else false.

If a node with the same key is already present inside the tree, the insert operation will simply return false without performing any action. This design choice has been taken in order to keep the implementation as simple as possible.
